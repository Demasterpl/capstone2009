Detailed Description

The class that loads, runs and handles events of the Shopped main GUI.

Parameters:
CurrentFileName The absolute path to the image file loaded.
CurrentImage Holds the state of the current image in the GUI.
TempImage Holds temporary states of the image in the GUI (i.e. zoom).
Zoom How much the image is zoomed in the GUI (1.0 == 100%).
DegreesRotated Holds the value for how much the image is rotated in the GUI (in degrees).
ImageRotate Instance of ImageRotate class to handle rotating the image.
FileOperation Instance of FileOperation class to handle opening/saving the image.
