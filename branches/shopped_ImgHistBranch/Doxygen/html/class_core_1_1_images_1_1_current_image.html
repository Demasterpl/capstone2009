Detailed Description

Holds the current state of the image in the Shopped GUI. Anytime the loaded image is modified (i.e. rotated, resized, etc.), this object gets updated to reflect the modification.

Parameters:
CurrentHeight Contains the height of the loaded image.
CurrentWidth Contains the width of the loaded image.
DegreesRotated Contains how many degrees the image is currently rotated.
